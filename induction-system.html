Introduction to Induction System

What is an Induction System?

The Induction System is a core feature of Jade Dynasty Lumen Server that promote the game and the server by encouraging players to introduce friends to the fantastic world of Jade Dyansty Lumen.

The system creates mentor-mentee relationships that benefit both parties. Inductors guide inductees through game mechanics, quests, and community norms, while inductees receive personalized assistance.

This system helps reduce the learning curve for newcomers and strengthens the overall community by fostering connections between players of different experience levels.

Side Note(s): Please do not confuse this with the Master & Disciple System. It is a different feature.

Inductor vs Inductee

Understanding the roles and responsibilities of both inductors and inductees is key to the system's success.

Inductor (Mentor)

  • Experienced player with comprehensive game knowledge (Level 60 and above required)
  • Provides guidance and answers questions
  • Helps inductee navigate game systems
  • Introduces inductee to community resources

Inductee (Mentee)

  • New or returning player seeking guidance
  • Commits to learning and active participation
  • Gains accelerated progression through guidance

Benefits

The induction system provides significant advantages for all participants and the community as a whole.

For Inductees

  • Faster learning curve and progression
  • Personalized guidance from experienced players
  • An Induction Pack (Brocade Box) which contains various useful rewards. A perfect starter kit for inductees.

For Inductors

  • Satisfaction of helping new players
  • Enhanced reputation within the server
  • Receive EXP and Bonus Jadens based on Inductee(s) progress

For the Community

  • Improved player retention
  • Knowledge sharing and preservation
  • More active and engaged player base
  • Healthier game ecosystem

How to Get Started

How to Become an Inductee

Becoming an inductee is a straightforward process.

Requirement(s)

  • New account (with no character created yet)

    • A clean slate account

  • Unfortunately, you can't become an inductee if you already have character(s) created

Steps to Become an Inductee

  1. Get an induction ID from your friend who introduce you to the game or find an experienced player willing to be your inductor (Level 60+). You can find them on discord before you create a character on your account.
  2. If you don't, feel free to use mine!

    3. Induction ID (for Classic)

    Use this code to register in the induction system:

    0020031X060200076001600Z1cd21f
  3. Once you choose your desired class/faction and are at the screen where you are going to customize your character, you will see the Inductor ID input box.

    4. On your left for human class (Classic)

    On your right for athan class (Classic)

  4. If it is Bound, it means your account already have an inductor. Otherwise, input (paste) your inductor ID and create your character(s).
  5. Once you enter the game, you can collect your induction pack from NPC Induction Agent.

    6. For Human Class - Sunstream City [185, 175]

    For Athan Class - Dragon Palace [265, -450]

  6. You will also get a meassage prompt to add you inductor as friend.
  7. Begin your journey with guidance from your inductor.

FAQ

  1. I already created a character without using any Induction ID. Where or how can I input it now?

    2. Unfortunately, you can't.

  2. Can I delete my character now (go back to a clean slate) to input the Induction ID?

    3. Yes, but you will lose all progress

  3. I already input someone's Induction ID, but he/she isn't active/helpful etc. Can I change my inductor ID to someone else?

    4. Unfortunately, NO. Even if you delete your character, it still won't be considered a clean slate once an Induction ID had been input before.

Wisdom from those who walked the path of immortal

  1. If you are new to JD or Lumen, it is inevitable that certain help will be needed
  2. Find an active inductor that is easily accessible (playing at the same time zone)
  3. Find an inductor that is willing to offer helps and assistances

How to Become an Inductor

Becoming an inductor requires meeting specific criteria and provides exclusive rewards for helping new players.

Requirement(s)

  • Character level 60 and above

Steps to Become an Inductor

  1. Talk to your respective Induction Agent once you are Level 60 and above.

    For Human Class - Sunstream City [185, 175]

    For Athan Class - Dragon Palace [265, -450]

    Select Get Induction Info

    Example of Induction Info

  2. Copy your Induction ID and forward it to your friend(s).

Inductor Responsibilities

  • Provide guidance on game mechanics and systems
  • Help with difficult quests and dungeons
  • Introduce inductees to community resources
  • Answer questions and provide support
  • Maintain a positive and helpful attitude

Claim Inductor's Rewards

  1. Talk to your respective Induction Agent.

    For Human Class - Sunstream City [185, 175]

    For Athan Class - Dragon Palace [265, -450]

    Select View Induction Info

    Offline inductees will not be shown on the list

  2. You will see a list of all your inductee(s) and click on the button Acquire to claim any bonus exp or jaden(s).
